# Dungeon
Dungeon game using Rust and SDL2

## Getting started

* Ensure Rust is installed: https://rustup.rs/
* Ensure SDL2 libraries are installed: `brew install sdl2 sdl2_image sdl2_ttf` (macOS)
* `cargo run`

## Controls

* Arrow keys: move
* Numbers 1 to 8: change character
* Esc: quit
